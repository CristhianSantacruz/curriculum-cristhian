Microservice Ecommerce

Implementamos un microservicio con Quarkus, utilizando health checks, tolerancia a fallos y métricas para monitorear y optimizar el rendimiento. Incluimos autorización, registro de usuarios, gestión de carritos y comentarios, y calificación de productos. Utilizamos REST clients para comunicación entre servicios y Panache Hibernate para acceso a la base de datos. La autenticación JWT garantiza la seguridad en las comunicaciones.

AdoptedPaw

Es un proyecto que facilita la adopción de mascotas a usuarios interesados. Ofrece la posibilidad de registrar una mascota para su adopción, proporcionando información detallada como edad, tamaño y una imagen cargada por el usuario para que otros puedan visualizarla. Además, permite a los usuarios interesados en adoptar establecer comunicación con el dueño actual de la mascota para iniciar el proceso de adopción.
